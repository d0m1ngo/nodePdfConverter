# PdfService

Foobar is a Python library for dealing with word pluralization.

## Installation

Use the package manager [yarn](https://yarnpkg.com) to install dependencies.

```bash
yarn install
```

## Prerequisite
```
node >= 7.x
graphicsmagick
ghostscript
mongoDb installed
 ```

## Usefull information

u could change /config/config.js for your aws urls, bucket and etc.
